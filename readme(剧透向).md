剧情：

根据名字不难发现我试图模仿《安德的游戏》。主角以为自己在挑战魔物（红色），自己是蓝色，但实际上这是一个设计好的《游戏》。主角作为魔王的候选者之一被召唤到了这个世界。他所击杀的，都是真正的人类。

关卡大概设计为四个大陆。主角需要前往四张地图寻找“击败魔王的圣器”。但最终它们实际上是成为魔王的信物。

在收集之后，我打算模仿一下《SANABI》的经典选择：

给出一张地图（一个短走廊），主角在中间，一侧红，一侧蓝。

选择红代表正式成为魔王，在与人类大战之后解锁结局：成为魔王或者被人类讨伐。

选择蓝代表回归人类：

如果你在收集“信物”的时候击杀了过多的人类，则人类无法接受你。解锁Bad Ending.

否则，红蓝转换。主角的颜色变为红色，去击杀更多的蓝色魔物，解锁结局：成为勇者或者被魔物讨伐。
